# Nsaq Agent Router

Use the correct agent role depending on the task.

## Use backend-agent when:
- Designing APIs
- Editing schema
- Building services
- Handling auth
- Building provider integrations
- Implementing logs / alerts backend

## Use frontend-agent when:
- Building pages
- Designing dashboard sections
- Handling i18n / RTL
- Improving UX/UI

## Use optimizer-agent when:
- Writing optimization rules
- Adjusting budget logic
- Changing bidding logic
- Reviewing thresholds / guardrails

## Use review-agent when:
- Reviewing pull requests
- Reviewing architecture
- Checking quality
- Validating multi-tenant safety

## Use product-agent when:
- Unsure whether feature belongs in phase 1
- Prioritizing roadmap
- Deciding between alternatives
- Preventing scope creep
